# MyOS VM系统控制台终端演示脚本

## 基本操作演示

1. 启动系统
./MyOS_VM_Test.exe

2. 选择交互式控制台模式 (输入 1)

3. 查看帮助
help

4. 查看系统状态
status

## VM管理演示

1. 创建x86 VM
vm create x86 x86_test.bin

2. 创建ARM VM  
vm create arm arm_test.bin

3. 创建x64 VM
vm create x64 x64_test.bin

4. 查看VM列表
vm list

5. 启动第一个VM
vm start 1

6. 运行10条指令
vm run 1 10

7. 查看VM详细信息
vm info 1

8. 暂停VM
vm pause 1

9. 恢复VM
vm resume 1

10. 停止VM
vm stop 1

## 调度器演示

1. 启动调度器
sched start

2. 将VM添加到调度队列
sched add 1 10

3. 绑定VM到核心2
sched bind 1 2

4. 查看调度器统计
sched stats

5. 解绑VM
sched unbind 1

6. 停止调度器
sched stop

## 性能监控演示

1. 启动性能监控
perf start 1

2. 运行VM
vm run 1 20

3. 停止性能监控
perf stop 1

4. 查看性能报告
perf report

## 批量操作演示

1. 创建多个VM
vm create x86 test_payload.bin
vm create arm test_payload.bin
vm create x64 test_payload.bin

2. 启动所有VM
vm start 1
vm start 2
vm start 3

3. 并行执行
vm run 1 5
vm run 2 5
vm run 3 5

4. 查看所有状态
vm list

## 清理操作

1. 删除VM
vm delete 3
vm delete 2
vm delete 1

2. 退出系统
exit

## 自动化测试

运行自动化测试套件:
./MyOS_VM_Test.exe
选择选项 2 (Automated Test Suite)

运行压力测试:
./stress_test.exe
